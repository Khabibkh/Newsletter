# Newsletter sign up
* Designed Frontend using **BootStrap 5** with **HTML5**, **CSS** and **JavaScript**.
* Build Backend using **NodeJS**.
* Used **[MailChimp API](https://mailchimp.com/)** for saving contact infos of customers who signed up for Newsletter.
